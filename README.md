# MkDocs Template

Several extensions have already been set up.

## Setup

```
git clone [to repo]
py -3.9 -m venv .venv
pip install -r requirements.txt
mkdocs serve

```

for mac to activate venv.  
```
source .venv/bin/activate
```

## Recommended extensions for VSCode

- Markdown All in One
- Markdown Image
